To install HinduFlipClock on your Android device:

1. On your Android phone/tablet, go to Settings ? Security (or Privacy) and enable Install unknown apps (or "Allow from this source")
2. Open this link on your device: [paste Google Drive link]
3. Tap Download
4. Once downloaded, tap the file to install
5. If prompted, tap Install anyway
